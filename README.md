LinguaCards

LinguaCards — это Android-приложение для изучения иностранной лексики с использованием карточек.
Приложение позволяет создавать наборы слов, проходить тренировочные сессии, отслеживать результаты и повторять материал в удобной форме.

Проект разработан в рамках курсовой работы и ориентирован на автономную работу без постоянного подключения к интернету.

Основной функционал:
регистрация и авторизация пользователя;
сохранение пользовательской сессии (повторный вход не требуется);
создание, редактирование и удаление карточек;
объединение карточек в наборы;
тренировка с использованием карточек;
свайпы карточек (влево / вправо) для оценки знания;
разворот карточки с анимацией;
подсчёт результатов тренировки;
возможность отмены последнего действия;
озвучивание термина с помощью Text-to-Speech;
хранение данных в локальной базе данных.

Используемые технологии:
Язык программирования: Kotlin
Среда разработки: Android Studio
Архитектура: Activity-based (без MVVM)
База данных: Room (SQLite)
Асинхронность: Kotlin Coroutines
UI: XML, Material Components
Озвучивание: Android TextToSpeech API

Требования к системе:
Android Studio: Hedgehog или новее
Android SDK: 34
Минимальная версия Android: Android 7.0 (API 24)

Версии Gradle:
Gradle: 8.6
Android Gradle Plugin: 8.4.0
Эти версии используются в проекте и рекомендуются для корректной сборки.

Установка и запуск проекта

1. Склонировать репозиторий:
git clone https://github.com/username/LinguaCards.git
2. Открыть проект в Android Studio:
File → Open
выбрать папку проекта
3. Дождаться завершения синхронизации Gradle.
4. При необходимости:
установить недостающие SDK через SDK Manager;
подтвердить использование указанных версий Gradle.
5. Запустить приложение:
выбрать эмулятор или физическое устройство;
нажать Run.

Все данные (пользователи, карточки, наборы, результаты тренировок) сохраняются локально с использованием Room.
